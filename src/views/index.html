Renovo Solutions UI Toolkit

This UI toolkit is a highly-modular design system for rapid web page development. It contains different materials that can be assembled into more complex page layouts.

This guide contains real working examples, code snippets, documentation, and style guidelines.

Usage

Start by including the toolkit files in your page:

<link rel="stylesheet" href="assets/toolkit/styles/toolkit.css">
<script src="assets/toolkit/scripts/toolkit.js"></script>

Typography

Renovo uses Lucida Grande as the primary typeface and Helvetica as a secondary typeface. Typographic elements are composed using a Modular Scale ratio of 4:3, or a Perfect Fourth. Character size, leading, measure, and spacing are all derived from this scale. Type is set in em units so that each typographic element scales proportionally.

You can see a more in-depth example of typesetting in the typography sampler.
